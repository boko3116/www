포토샵 단축키

  • 새 파일 만들기 : Ctrl + N
  • 불러오기 : Ctrl + O
  • 창 닫기 : Ctrl + W
  • 열려있는 모든 창 닫기 : Ctrl + Alt + W
  • 저장 : Ctrl + S
  • 다른이름으로 저장 : Ctrl + Shift + S
  • 복사해서 저장 : Ctrl + Alt + S
  • 이미지 인쇄 : Ctrl + P
  • 포토샵 종료 : Ctrl + Q

이미지 단축키

  • 레벨 메뉴 실행 : Ctrl + L
  • 커버 메뉴 실행 : Ctrl + M
  • 색조/채도 메뉴 : Ctrl + U
  • 컬러 밸런스 메뉴 : Ctrl + U
